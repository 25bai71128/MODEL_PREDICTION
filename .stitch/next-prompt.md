Modern landing page for the Model Recommendation Studio - a professional ML model selection and benchmarking platform.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Palette: Primary Background (#f4efe7 for warm canvas), Secondary Background (#fbf8f2 for cards), Panel Background (rgba(255, 255, 255, 0.84) for floating elements), Text Primary (#101828 for headings), Text Muted (#5b6472 for secondary text), Accent Primary (#0f766e for actions), Accent Gold (#b7791f for highlights)
- Styles: Rounded rectangles (16-22px radius), Soft diffused shadows (0 22px 50px rgba(15, 23, 42, 0.09)), Pill-shaped tabs and buttons

**PAGE STRUCTURE:**
1. **Header:** Sticky navigation bar with logo "Model Recommendation Studio", menu items (Upload Dataset, Analysis, Benchmark, Quality Check, Export), primary CTA button "Get Started"
2. **Hero Section:** Large headline "Find the Perfect ML Model for Your Data", subtext explaining the dual AI + live benchmark approach, prominent upload area with drag-and-drop, key features callouts in mini-cards
3. **How It Works:** Step-by-step process explanation with numbered sections and icons
4. **Features Grid:** Grid of feature cards highlighting meta-model recommendations, live benchmarking, data quality checks, exportable reports
5. **Footer:** Links to documentation, API, GitHub, contact information
